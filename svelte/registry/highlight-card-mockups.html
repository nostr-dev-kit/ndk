Proposed Component Structure

Component Name: HighlightCard

Variants:

  • default / feed - Large book-page style with author header and actions
  • compact - List view with left accent bar and minimal metadata
  • grid - Square cards for grid layouts with author below
  • minimal - Inline blockquote style (NEW proposal)

Key Features:

  • Displays highlighted text with optional surrounding context
  • Shows source (web URL, Nostr article, or event)
  • Dynamic font sizing based on content length
  • Optional author information
  • Optional engagement actions (comments, reactions, shares)
  • Fully composable with Root + Children pattern
