Custom Implementation

Use the createTimelineFeed builder to create fully custom feed implementations.

import { createTimelineFeed } from '@nostr-dev-kit/svelte'; const feed = createTimelineFeed(() => ({ filter: { kinds: [1], limit: 50 }, enableRealtime: true }), ndk); // Access reactive state feed.events // $state array of NDKEvent[] feed.isLoading // $state boolean feed.hasMore // $state boolean // Methods await feed.loadMore() await feed.refresh()

Component API

TimelineFeed

Prop Type Default Description
ndk NDKSvelte ‚Äî NDK instance (optional if provided via context)
filter required NDKFilter ‚Äî Filter for events to display
limit number 50 Initial events to load
enableRealtime boolean true Subscribe to new events
enableInfiniteScroll boolean true Load more on scroll
variant 'default' | 'compact' | 'grid' 'default' Visual style variant
showActions boolean true Show reply/repost/like/zap actions
class string '' Custom CSS classes

Events

Event Payload Description
eventclick { event: NDKEvent } Fired when an event card is clicked
loadmore { count: number } Fired when more events are loaded
error { error: Error } Fired on errors

Property Type Description
