Infinite scroll feeds with real-time updates, filtering, and multiple layout options. Build Twitter-like, Instagram-like, or custom feed experiences in minutes.

Component API

Prop Type Default Description
ndk NDKSvelte — NDK instance (optional if provided via context)
filterrequired NDKFilter — Filter for events to display
limit number 50 Initial number of events to load
enableRealtime boolean true Subscribe to new events in real-time
enableInfiniteScroll boolean true Automatically load more on scroll
variant 'default' | 'compact' | 'grid' 'default' Visual style variant
showActions boolean true Show reply/repost/like/zap actions
class string '' Custom CSS classes for styling

Builder API

Use the createTimelineFeed builder for full control over feed behavior.

import { createTimelineFeed } from '@nostr-dev-kit/svelte';

const feed = createTimelineFeed(() => ({
  filter: { kinds: [1], limit: 50 },
  enableRealtime: true
}), ndk);

// Access reactive state
feed.events       // $state array of NDKEvent[]
feed.isLoading   // $state boolean
feed.hasMore     // $state boolean

// Methods
await feed.loadMore()
await feed.refresh()
