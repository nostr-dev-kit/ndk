Data Flow

  1. Component receives relayUrl prop
  2. Relay.Root fetches NIP-11 metadata from {relayUrl}/.well-known/nostr.json
  3. Metadata stored in context, available to all child primitives
  4. Each primitive reads specific fields from context
  5. User interactions (bookmark) publish events via NDK
