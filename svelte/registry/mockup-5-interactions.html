Available Event Handlers

onclick
Fired when card is clicked. Use for navigation or selection.
onclick={(e) => selectRelay(relayUrl)}
onmouseenter
Fired when mouse enters. Use for preloading or tooltips.
onmouseenter={() => prefetchRelayData()}
onmouseleave
Fired when mouse leaves. Clean up hover effects.
onmouseleave={() => hideTooltip()}
onfocus
Keyboard focus. Important for accessibility.
onfocus={() => announceToScreenReader()}

Error Handling

Relay Offline
Shows offline indicator, grays out card, disables interactions
NIP-11 Fetch Failed
Falls back to URL-based display, shows minimal card
Bookmark Failed
Reverts button state, shows error toast, logs to console
Invalid URL
Shows error card with message, provides correction hint

Custom Events

Relay cards emit custom events you can listen to:

// Listen for relay selection card.addEventListener('relay:select', (e) => { console.log('Selected:', e.detail.url) }) // Listen for bookmark changes card.addEventListener('relay:bookmark', (e) => { console.log('Bookmarked:', e.detail.bookmarked) }) // Listen for metadata loaded card.addEventListener('relay:loaded', (e) => { console.log('Metadata:', e.detail.nip11) })
