Build The Complete Flow

Wire up a full social feed in 30 lines

// Install the components you need npx jsrepo add components/event-card npx jsrepo add components/reaction-button npx jsrepo add components/zap-button npx jsrepo add builders/create-reaction-action npx jsrepo add builders/create-zap-action // Use them in your app <script lang="ts"> import { EventCard, ReactionButton, ZapButton } from '$lib/components'; import { createReactionAction, createZapAction } from '$lib/builders'; // Create reactive actions const reactionAction = createReactionAction(() => ({ target: event })); const zapAction = createZapAction(() => ({ target: event })); </script> // Render complete event with all interactions <EventCard.Classic {event}> <EventCard.Header {event} /> <EventCard.Content {event} /> <EventCard.Actions> <ReactionButton {reactionAction} /> <ZapButton {zapAction} /> </EventCard.Actions> </EventCard.Classic> // That's it.
