Relay Card

A versatile card component for displaying Nostr relay information

The Relay Card component provides multiple ready-made variants for displaying relay metadata, status, and actions. Built using composable headless primitives, it supports NIP-11 metadata fetching, connection status indicators, and interactive relay management.

Each variant is optimized for different use cases—from compact lists to detailed portraits—while maintaining full customization through the underlying primitive layer.

Interactive breakdown of relay card structure

Relay.Icon

Displays the relay's icon or favicon. Falls back to a generic relay icon if not available. Supports custom size and styling.

url size fallback

Relay.Header

Contains the relay name and URL. Automatically fetches and displays relay name from NIP-11 metadata.

name url showUrl

Relay.Metadata

Displays relay metadata badges including supported NIPs, relay type, and other information from NIP-11.

supportedNips software version

Relay.Description

Shows the relay description from NIP-11 metadata. Supports truncation and expansion for long descriptions.

maxLength expandable

Relay.Actions

Container for action buttons like connect, disconnect, copy URL, or add to relay set. Fully customizable.

onConnect onDisconnect onCopy

Components

Ready-made relay card implementations

RelayCardPortrait

A vertical portrait-oriented card displaying comprehensive relay information including icon, metadata, description, and actions.

Usage 
<script>
  import { RelayCardPortrait } from '@nostr-svelte/ui';
  import NDK from '@nostr-dev-kit/ndk';

  const ndk = new NDK();
</script>

<RelayCardPortrait
  {ndk}
  url="wss://relay.damus.io"
/>
