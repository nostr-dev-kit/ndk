Relay Card

A versatile card component for displaying Nostr relay information

The Relay Card component provides multiple ready-made variants for displaying relay metadata, status, and actions. Built using composable headless primitives, it supports NIP-11 metadata fetching, connection status indicators, and interactive relay management.

Each variant is optimized for different use cases—from compact lists to detailed portraits—while maintaining full customization through the underlying primitive layer.

Usage 
<script>
  import { RelayCardPortrait } from '@nostr-svelte/ui';
  import NDK from '@nostr-dev-kit/ndk';

  const ndk = new NDK();
</script>

<RelayCardPortrait
  {ndk}
  url="wss://relay.damus.io"
/>
npx shadcn-svelte@latest add relay-card-portrait

This command will add the RelayCardPortrait component and all its dependencies to your project.
