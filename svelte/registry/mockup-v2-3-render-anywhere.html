The Pattern: Context-Aware Rendering

Components don't just render. They understand where they are and adapt.

Highlights know if they're standalone or inline. Reactions know if they're in a feed or modal. User components know if they're headers, cards, or mentions.

One set of primitives. Infinite rendering contexts. All handled automatically.
